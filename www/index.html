Profiles 我的信息

这是什么

这是基于Nebulas的Profiles，用于储存您的昵称，头像，简介，用于各个网站使用。你不用为各个不同的网站分别设置自己的信息，只需要设置一次全网即可使用。
就像很多区块链网站发布信息的时候还需要现写一个昵称，这种是不合理的，但是要自己做一个用户信息系统有很麻烦。
所以才有了这个项目。

如何使用

在页面中引入profiles.js或profiles.min.js.简单的调用名片如下,会填入用户名,并且展示一个简单的名片。 

<a class="profile-link" data-wallet="n1atVgm..." href="#">n1atVgm...</a>
...
<script src="js/nebulas.js"></script>
<script src="js/profiles.js"></script>
            

<script>
Profiles.init();
// ... or...
Profiles.elem(document.getElementById('...'));
</script>
n1atVgmUM2m2bWMMaDTDhn4T3w2JHnVseTP n1TRakxM5EyF29J2mhhuujhq44Pv663rfQr
也可以单独获得用户信息 

Profiles.fetch(wallet_address).then(function(res){
    console.log(res);
}).catch(function(err) {
    console.log(err);
});
只需要做一个头像img的情况如下. 

<img class="avatar" data-wallet-image="n1atVgmUM2m2bWMMaDTDhn4T3w2JHnVseTP"/>
以上,那么现在在右上角"我的信息"中修改你自己的名片，并保存就可以了。
